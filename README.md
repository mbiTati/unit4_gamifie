# Unit 4 — Database Design & Development (Gamifié)

## 🎮 Cours gamifié MySQL — BTEC Level 4

**Structure d'un module :** Chaque module suit un cycle en 6 phases :
1. **Théorie** — Slides interactives + PPTX enseignant
2. **Mini-jeu visuel** — Jeu graphique pour découvrir le concept SANS coder
3. **Quiz** — Questions intégrées après chaque section théorique
4. **Code guide + Exercice** — Code guidé pas à pas + exercice autonome
5. **Correction + Défi bonus** — Correction complète + défi optionnel
6. **Mémo déblocable** — Fiche récapitulative (récompense)

## 📂 Structure du repo

```
docs/          → Plans, documents DOCX
jeux/          → Mini-jeux HTML/JS (déployables Vercel)
sql/           → Scripts SQL (MySQL)
pptx/          → Présentations enseignant
assets/        → Images, ressources partagées
```
