Remplissez la matrice des DF

Cliquez sur les cases pour indiquer les dependances fonctionnelles entre les donnees de SwissSound Studios. Si A determine B, cliquez la case (ligne A, colonne B).

Rappel : A → B signifie « connaitre A suffit pour determiner B de facon unique ».
Exemple : id_artiste → nom_artiste car un id identifie un seul artiste.
Contre-exemple : nom_artiste → id_artiste est FAUX car il peut y avoir des homonymes.
Les sources (lignes avec des 1 mais colonne sans 1) deviendront les futures cles primaires.
