Animation SELECT + WHERE

Observez comment WHERE filtre les lignes une par une. Seules les lignes correspondantes s'allument.

SELECT nom_modele, prix_vente FROM modele WHERE categorie = 'Running';

Animation INNER JOIN

Deux tables se rapprochent. Les lignes avec la même clé se connectent et fusionnent.

SELECT m.nom_modele, ma.nom_marque FROM modele m JOIN marque ma ON m.id_marque = ma.id_marque;

Animation INSERT

Une nouvelle ligne apparaît et glisse dans la table.

INSERT INTO client (nom, prenom, ville) VALUES ('Zeller', 'Marc', 'Berne');

Animation DELETE

Les lignes correspondant au WHERE disparaissent progressivement.

DELETE FROM client WHERE ville = 'Sion';

Animation GROUP BY

Les lignes se regroupent par catégorie, puis l'agrégation calcule le résultat.

SELECT categorie, COUNT(*), ROUND(AVG(prix_vente),2) FROM modele GROUP BY categorie;
